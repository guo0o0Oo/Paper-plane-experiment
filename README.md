# Paper-plane-experiment
An exploratory learning experiment investigating the flight path of paper airplanes, their center of gravity, and wing dimensions.  
This research project was a collaborative effort completed by cht, rzr, yjy, and myself.  
As part of this research-based learning project, yjy, cht, and I collaborated on designing and carrying out the experiment. I was also in charge of handling the experimental data.During data processing,I utilized the YOLOv8 object detection model and libraries such as OpenCV within a Python environment for creating data files.At last,the position and attitude images of paper planes produced under various conditions serve as the final output.  
The transaction related to physical modeling and analysis data has been assigned to rzr and is currently being processed.
